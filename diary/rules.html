What Why and How of Bilzaa Notes

This repository is my notes folder for every thing. I have been writing notes for years, however I keep loosing them since they do not have one simple home. This is a one-stop solution for all my notes taking needs. Also these notes are shared online with other people so that they can take help from it.

The main purpose of this endevour is:

  1. Store all and everything specially written text, images, snippets etc. Any thing that can be saved as html file should be saved.
  2. It will not use a database since it has to be textbase and has to be kept simple (I can however use comma seperate files or json files, such that we can use then as replacement of database)
  3. KEEP IT simple.
  4. Use normal webpages with javascript , html and CSS
  5. Use of web-components is allowed
  6. Any change in the codebase that prevents me from just moving the folder from one place to another is prohibited
  7. In the past I have created a code base system. That became too complicated so now all i will use is simple webpages and a manually build home page, nothing more no react no database
  8. The word code base means that this will serve as a base for any future complex and modern system but this itself will be very simple and non-technical.

Here are the Rules for This Codebase System

  1. Keep it very simple. By simplei mean I can add javascript and CSS but no server side database etc etc. Any thing that prevents me from copy pasting this folder to another place it prohibited. No PHP, react etc just HMTL, javascript and CSS.
  2. How do I know if the system is simple engough? Well here are the criteria: 1: You should be able to copy and paste this folder any where without any tooling or server setup and this should work fine. 2: This should be able to be dragged onto a server and become a ready made blog (quick and dirty blog)
  3. Improve Forward, Support Backword: Never add new things to old pages, once a page is written just let it be, if there is some improvement just add it to the future pages.
  4. This is not a folder but internet Just like all pages on internet do not look same and work the same similarly this folder is not suppose to be uniform, it just has to be linked, every page must have atleast one link out of it, the index.html can have millions of links and will serve as a database.
