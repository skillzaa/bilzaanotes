Tail Wind CSS Important Classes

Padding For padding we use p-10, p-1 etc. P stand for padding, we can use px-10 and py-10 etc for padding in X and Y direction.

Margin: Just like padding we can use m-10, mx-10 etc, an important class is mx-auto which is used for centring objects (mx stand for margin X).

Keep in mind every class that we mention here has a large number of variations, we do not need to mention every variation here we just have to remember a class and then we can look up its other variations.

Shadow: For shadow, shadow-outline etc

rounded: For rounding cornors, some variations are rounded-full

Border: we use border for border, some variations are border-bottom/left/right etc, border-l-2 etc. Following are some examples of borders (for some reason to show border one has to add border-solid). Border left (border-l-8) is just upto 8. border-dotted works where as border-double does not work.

Backround Colors: we use bg-red-100, there are many colors and they range from 100 to 900 thus we can make them light and strong.

Psedo Classes: We have active, hover and focus psedu classes which we can use just with semi colon eg hover:bg-red-100 etc.

Screen Sizes: Just like Bootstrap we have 4 screen sizes sm,md,lg and xl.

Container: We do have container class but it does not auto center the text etc.

font: we use font-bold, font-semibold etc

text: we use text-lg, text-xl to increase the size of text. To change text color we usr text-white etc

Font Family: we can use font-seirf etc to set font family.The beauty of tailwind is once you get the hang of the classes you can make any design without using any CSS and just using CSS classes.

Width: we use w-?? for width w-1/2 is half width. w-screen is view port width.
